:: directly used ::

assets and tools that are actively loaded or used on this site.

sadhost.neocities.org ~826 tiles the massive archive of tiled backgrounds used on every page. animated gifs, pixel art, textures — all of it.
cyber.dabamos.de/88x31 4540 badges the definitive archive of 88×31 pixel badges. the footer row of little stickers on every page.
blinkies.cafe 283 blinkies animated blinkie GIFs in the sidebar. each one is a tiny piece of y2k web nostalgia.
VT323 (google fonts) 1 font the monospace terminal font used for headers and the site title. looks like it belongs on a CRT.
github pages — free static site hosting. no servers, no backend, no build step — just files.
:: inspiration ::

sites and communities that shaped the aesthetic and approach of this site.

neocities.org — the web revival hosting platform. basically the reason any of this exists. bring back personal homepages.
sadgrl.online — webmastery resources, layout guides, and old web tutorials. great starting point if you want to build your own.
melonland.net — indie web forum and community. good place to find people who still care about making things on the internet.
fructisfans / links — a well-curated hub of old web resources — backgrounds, blinkies, guestbooks, webrings, tools. packed with stuff.
